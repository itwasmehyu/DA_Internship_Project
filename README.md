# Data Analysis Internship Project 

## 📌 Giới Thiệu
Dự án này tập trung vào việc phân tích và xử lý dữ liệu [Ví dụ: hành vi người dùng/doanh thu quảng cáo] nhằm tìm ra các Insight quan trọng phục vụ cho việc ra quyết định kinh doanh. Đây là dự án trong khuôn khổ chương trình thực tập.

## 📊 Dữ Liệu (Dataset)
Do giới hạn kích thước file của GitHub (Max 100MB), các tập dữ liệu thô và dữ liệu đã qua xử lý (lên tới 4GB) được lưu trữ trên Google Drive.

* **Link tải Dataset:** [https://drive.google.com/drive/folders/1KCPMxO7tY0-DpkPufn7MeXC1Ly9Jne2n?usp=sharing]
* **Các file chính:**
    * `AD_REVENUE.csv`: Dữ liệu về doanh thu quảng cáo.
    * `ENGAGEMENT.csv`: Dữ liệu về tương tác người dùng.
    * `merged_user_data.csv`: Dữ liệu tổng hợp sau khi xử lý (Cleaned & Merged).

> **Lưu ý:** Sau khi tải về, vui lòng giải nén và đặt vào thư mục `/Dataset` để code có thể chạy đúng đường dẫn.

## 🛠️ Công Nghệ Sử Dụng
* **Ngôn ngữ:** Python (Pandas, Numpy, Matplotlib, Seaborn)
* **Công cụ:** Jupyter Notebook / VS Code
* **Kỹ thuật:** Data Cleaning, EDA (Exploratory Data Analysis), Data Merging.
